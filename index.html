BIO

From belonging to building software that connects people.

During my adolescence, moving to the UK made me feel somewhat out of place, and I often wondered how I could better integrate into society. This curiosity sparked my interest in human interactions and the challenges people face in connecting with one another.

After graduating from university, these reflections evolved into ideas that, combined with my software skills, transformed into potential applications. This vision became the driving force behind developing my technical expertise and exploring the vast possibilities of software development. Now, with AI accelerating the realization of this vision, I am actively exploring the frontiers of AI, knowing that the applications I develop could have a transformative impact on humanity.

This journey continues to inspire me to leverage technology to bridge gaps in human interaction and create meaningful, real-world impact.

Echelon Estates

A luxury property listings website built with Advanced Custom Fields (ACF) and Elementor to create the theme and templates. Yoast SEO was used to improve the site's SEO, Dynamic Tags helped keep content consistent, and property listings were imported using CSV imports.

Basket & Co

An e-commerce website for everyday essentials and unique finds, with product categories, search, cart, and checkout built with WooCommerce. PayPal and Stripe test payments are configured via WooCommerce checkout, using Blocksy, Elementor, and a Custom Post Type for structure and layout, with Yoast SEO and Site Kit (Google Analytics) for search optimisation and performance tracking.

Keywords: WooCommerce, Elementor, Yoast SEO, Site Kit, Blocksy, Custom Post Type

FilmFlick

FilmFlick is a Vue.js-based movie discovery platform for browsing, searching, and exploring films with real-time data. Users can filter by genre, view ratings, watch trailers, and manage watchlists through an interactive interface. Built with Vue 3, Vue Router, and Bootstrap 5 for responsive design. Integrates The Movie Database (TMDB) API for movie data, uses Vue's Composition API and provide/inject pattern for state management. Features component-based architecture with reusable UI elements and mobile-responsive design across all screen sizes.

Keywords: Vue, Vue Router, Bootstrap 5, TMDB API

Macros Calculator website

React-based macro tracker for daily carbs, protein, and fats. Users can sign up, log in, and save entries to Firebase Realtime Database. Built with React 17, Bootstrap 5, React Bootstrap, and Styled Components. Uses REST, React Router DOM, and Context API for navigation and state. Secure authentication with auto-logout after 30 minutes.

Keywords: React 17, Styled Components, Bootstrap 5, REST

Suits and Styles

Taste Travel
